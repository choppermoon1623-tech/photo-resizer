# 写真一括リサイズ

ブラウザだけで動くローカル用の写真一括リサイズツールです。画像は外部へ送信されません。

## 使い方

1. `index.html` をブラウザで開きます。
2. 画像をドラッグ＆ドロップするか、クリックして複数選択します。
3. 最大幅、最大高さ、出力形式、画質を設定します。
4. `一括変換` を押します。
5. 個別保存、または `ZIPで保存` でまとめて保存します。

## 対応内容

- 複数画像の一括処理
- 縦横比を保ったリサイズ
- JPEG / PNG / WebP 出力
- JPEG / WebP の画質調整
- 小さい画像を拡大しない設定
- ZIP一括ダウンロード
